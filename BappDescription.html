Burp to Discord automatically forwards all issues to a Discord webhook as they are detected. This extension listens for issues from any source and delivers real-time notifications to Discord channels.

Features

  • Real-time Discord notifications for all reported issues
  • Configurable user mentions with Discord user ID integration
  • Smart truncation of oversized requests and responses to stay within Discord's message limits

Usage

  1. Navigate to "Settings → Extensions → Burp to Discord"
  2. Create a Discord webhook: right-click your channel, select "Edit Channel → Integrations → Webhooks", and copy the webhook URL
  3. Paste the webhook URL into the "Discord Webhook URL" field
  4. Optional: Enable user mentions by providing your Discord User ID (enable Developer Mode in Discord, then copy your user ID from your profile)
  5. Configure whether to include request/response data in notifications

When an issue is detected, you'll receive a Discord notification containing the issue title, affected URL, and details. If request/response inclusion is enabled, the extension will send the HTTP traffic in follow-up messages, automatically truncating large payloads to prevent delivery failures.
